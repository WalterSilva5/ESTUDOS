## crates: 
  binary: aplicações com pacote main que podem ser executadas
  library: pacotes e utilitaros que podem ser utilizados por outros crates (tanto binary quanto library)

o cargo é utilizado para gerenciar crates
comandos cargo:

    cargo new
    cargo build
    cargo run
    cargo clean
    cargo check
    cargo doc
